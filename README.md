## Generative AI Usage

For the tasks PlusOne, ReverseInteger, SortColors, and ValidPerfectSquare, generative AI was used as a tutoring and writing-support tool. It was used to clarify assignment expectations, sanity-check the testing rationale, help interpret JaCoCo and PIT results, and improve the structure and wording of the markdown reports.

The test design, bug identification, code fixes, and final decisions were made independently. All AI-assisted suggestions were manually reviewed and verified against the code, test runs, and coverage reports before being included. Full responsibility is taken for the final submitted solution.
